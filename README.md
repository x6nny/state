# state
A simple, yet versatile state class.

> [!NOTE]
> This is not a full state management library.
> This is only a state class.
---

# Features
- ```state:read()```
- ```state:write()```
- ```state:listen()```
- ```state:send()```
- ```state:enabled()```
- ```state:is()```
- ```state:isNot()```
